## 開発の目的

授業で文字数カウンターを使用する機会が多く、より使いやすい文字数カウンターアプリを作成したいと思ったから。

## 開発の始め方

```bash
# 任意の場所（例：D:\）にフォルダーを作成し、以下のコマンドを実行
git clone https://github.com/aoriikanosuke/moji-counter.git
cd moji-counter
npm install
npm run dev
```
